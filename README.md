# Focus Time

A minimal pomodoro timer with task tracking.

## Features

- Customizable focus and break durations
- Quick presets (25/5, 50/10, 90/15)
- Task checklist to track what you want to accomplish
- Early completion detection - finish all tasks early and get congratulated
- Audio notification when timer ends

## Usage

1. Enter tasks you want to accomplish (one per line)
2. Set focus and break times or use a preset
3. Click Begin to start
4. Check off tasks as you complete them
5. Complete all tasks early to end the session

## Run Locally

Open `pomodoro.html` in a browser, or serve with:

```
python3 -m http.server 8000
```

Then visit http://localhost:8000/pomodoro.html
